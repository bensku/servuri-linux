# servuri-linux
[bootc](https://bootc.dev/) builds of OpenSuse Tumbleweed for my homelab.
Take a look, copy if you find something of use, *don't* use these unchanged!

Inspired by awesome [bootcrew](https://github.com/bootcrew/mono) project!
The main difference is that this uses grub2 and
[bootupd](https://github.com/coreos/bootupd) instead of
systemd-boot for legacy BIOS boot support.
